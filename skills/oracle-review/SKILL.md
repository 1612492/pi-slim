# Oracle Review

Use this skill when delegating code review or architecture scrutiny to `oracle`.

## Good oracle tasks

- review a fixer's implementation
- identify correctness or maintainability risks
- evaluate tradeoffs
- suggest simplifications

## Rules

- Ask for findings ordered by severity.
- Prefer exact file references and concrete next moves.
- Use oracle as a reviewer/advisor, not as an editor.
