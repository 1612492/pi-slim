# Fixer Execution

Use this skill when delegating implementation to the `fixer` subagent.

## Good fixer tasks

- scoped feature work
- bug fixes
- targeted refactors
- test updates

## Rules

- Keep tasks bounded and specific.
- Prefer existing patterns over broader redesign.
- Ask fixer to report exact files changed and verification notes.
- Use `oracle` after fixer when you want a review or strategic second pass.
